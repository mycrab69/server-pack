# server-pack
server-pack for self-hosted server

## resource pack made by gamingbarn
This resource pack is a combination of 4 mods from [gamingbarn][https://modrinth.com/user/Gamingbarn]; those are:
* [Chems' gun][https://modrinth.com/datapack/chems-guns]
* [Pipe gun][https://modrinth.com/datapack/pipe-guns]
* [Modern gun][https://modrinth.com/datapack/modern-guns]
* [Gamingbarn's gun][https://modrinth.com/datapack/gamingbarns-guns]
